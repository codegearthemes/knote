== Description ==

Knote is a Google core web vitals optimised WordPress theme built with the latest web technologies. It is lightning fast, lightweight (less than 68KB on front-end) and highly customizable. Built with performance in mind, it comes with tons of performance enhancement features for unparalleled speed. Plus, it is schema.org friendly and SEO optimised to get top ranking in Google. Knote is perfect for business, portfolio, education, startup, agencies, digital marketing, travel, eCommerce shop (WooCommerce), magazine, coaching, affiliate marketing, non-profit, events, blog and most types of websites. It works flawlessly with popular page builders ( Elementor & Editor ), Yoast, RankMath and WooCommerce. Some other features include: Translation Ready, Responsive and starter templates. Looking for a Google core web vitals optimised WordPress theme with unmatched performance? Look no further. Try Knote theme today!

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
